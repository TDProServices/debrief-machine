# PROMPT 3: ADHD-FRIENDLY SESSION GUIDE

**Best for:** People who are ADHD or work with ADHD-friendly structures

**Output:** Brain-friendly documentation that doesn't overwhelm

- **Bold key points** (quick visual emphasis)
- **Clear boundaries** (start/end of session is clear)
- **Action-focused language** (what should I DO?)
