### Smart Versioning Protocol

1. Search: "debrief version project briefing"
2. Check chat history for "briefing v"
3. Apply logic:
    - No previous = v1.0
    - Minor changes = increment patch (v1.0.1)
    - New features = increment minor (v1.1.0)
    - Major changes = increment major (v2.0.0)
