# The Debrief Machine

**Version:** 3.0
**Status:** Ready for productization
**Purpose:** Comprehensive AI conversation analysis and project debrief system

---

## What This Is

The Debrief Machine is a battle-tested prompt system for conducting comprehensive project debriefs and conversation analysis using AI. It guides LLMs through a systematic 26-section analysis framework.

## Key Features

- **Multi-LLM Compatible:** Works with ChatGPT, Claude, Perplexity, and other major LLMs
- **Comprehensive Framework:** 26-section analysis covering technical, business, and strategic aspects
- **Version History:** Evolved from v1.0 (basic debrief) to v3.0 (full autonomous system)
- **Export-Ready:** Generates professional reports suitable for stakeholders

## Use Cases

1. **Project Retrospectives** - Analyze completed projects for lessons learned
2. **AI Conversation Mining** - Extract value from ChatGPT/Claude conversations
3. **Strategic Planning** - Identify patterns and opportunities across work
4. **Knowledge Transfer** - Document tacit knowledge for team members
5. **Product Development** - Analyze prototype sessions for product insights

## How to Use

1. Choose the appropriate version for your LLM
2. Copy the full prompt into your AI conversation
3. Provide the content to analyze (conversation, project docs, etc.)
4. Let the AI work through all 26 sections systematically
5. Review and export the generated debrief
