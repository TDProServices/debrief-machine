## üîÑ **Daily Bridge Workflow**

### **Morning Bridge (5 minutes):**
```
1. Check yesterday's Obsidian daily note
2. Write today's focus/agenda in notebook
3. Note any active project contexts
4. Start fresh notebook page with date + project list
```

### **Throughout Day:**
```
- Capture everything in notebook (meetings, ideas, sketches)
- Add ‚òÖ to promising items as you go
- Don't worry about organization - just capture
```

### **Evening Bridge (10 minutes):**
```
1. Count starred items in notebook
2. Transfer stars to Obsidian daily note
3. Quick sort: which active project does each belong to?
4. Add any immediate constellation observations
5. Archive completed notebook page
```

### **Weekly Constellation Review (15 minutes):**
```
1. Review week's transferred items
2. Group by domain (productivity, health, business, etc.)
3. Notice any new patterns or connections
4. Update domain index notes with insights
5. Plan next week based on constellation developments
```
